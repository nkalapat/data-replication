#After reorganizing some of the novel object recognition test data in a new excel sheet, I loaded it in using the read_excel() function. The novel object recognition (NOR) test is used to assess recognition memory and relies on the natural tendency of mice to explore a novel object over a familiar one. The time spent exploring a previously explored object (Old column) and time spent exploring a new object (New column) are recorded for each mouse. The mice are divided into two different groups, control and EtOH-treated. The percentage of time spent exploring the novel object, calculated as (time spent exploring novel object/total exploration time)*100, is recorded in the Ratio column. 
NOR_data <- read_excel("data/Raw Data for Data Replication Project.xlsx", sheet = 11)

#I used the group_by() function to analyze the data by experimental group (Control vs. EtOH). I calculated the average time spent exploring the new object, average time exploring the old object, standard deviation in exploration time of the new object, standard deviation in exploration time of the old object, and average percentage of time spent exploring the new object for each experimental group. The mean percentage values line up with what was calculated by the authors.
NOR_data %>%
  group_by(Group) %>%
  summarise(
    mean_novel = mean(New, na.rm = TRUE),
    mean_familiar = mean(Old, na.rm = TRUE),
    sd_novel = sd(New, na.rm = TRUE),
    sd_familiar = sd(Old, na.rm = TRUE), 
    mean_ratio = mean(Ratio, na.rm = TRUE)
  )

#I made a simple bar graph, which is what the authors of the paper did, to visualize the average percentage of time spent exploring the novel object by experimental group. I used the stat_summary function to graph the average percentage of time spent exploring the novel object, and to add error bars. I also added in the individual data points with geom_jitter(), which is what the authors did as well. 
NOR_data <- NOR_data %>%
  mutate(Ratio = as.numeric(Ratio))
  
ggplot(NOR_data, aes(x = Group, y = Ratio, fill = Group)) +
  stat_summary(fun = mean, geom = "bar", width = 0.5) +
  geom_jitter(width = 0.15, size = 2.5, alpha = 0.8) + 
  stat_summary(fun.data = mean_se, geom = "errorbar", width = 0.2) +
  labs(title = "Average Percentage of Time Spent Exploring the Novel Object", x = "Experimental       Group", y = "Average Percentage") + 
  theme_minimal()
